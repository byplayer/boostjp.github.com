無向グラフの 帯域幅 (bandwidth) は二つの隣接頂点の間の最大距離 で、頂点が構成単位間隔に置かれた線上で測定された距離である。別の言い方を すると、 もし無向グラフの頂点 G=(V,E) に各々 0 から |V| - 1 までの index[v] によって与えられる添え字が割り当てられているなら、G の帯域幅は

B(G) = max { |index[u] - index[v]|  | (u,v) in E }

である。

		const Graph& g,

グラフの i 番目の帯域幅 (i-th bandwidth) は i 番目の頂点 とその隣接のいずれかとの間の最大距離である。

Bi(G) = max { |index[i] - index[j]|  | (i,j) in E }

それで帯域幅 B(G) は i 番目の帯域幅 Bi(G) の最大値 として表すことができる。

B(G) = max { Bi(G)   | i=0...|V|-1 }
