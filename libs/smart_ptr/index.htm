Smart Pointer Library

このスマートポインタライブラリは 5 種類のスマートポインタクラステンプレートを含む。 スマートポインタは C++ の new で動的に割り当てられるメモリの管理を容易にする。 更に scoped_prt は他の方法で動的に割り当てられたメモリの管理を容易にする。
