BOOST_PP_ARRAY_DATA マクロは配列からタプルのデータを抽出する。

Usage

BOOST_PP_ARRAY_DATA(array)

Arguments

array
タプルに変換される配列。

Remarks

このマクロは配列データの一部分であるタプル部分に展開される。

Requirements

Header:  <boost/preprocessor/array/data.hpp>

Sample Code

#include <boost/preprocessor/array/data.hpp>

#define ARRAY (3, (x, y, z))

BOOST_PP_ARRAY_DATA(ARRAY) // (x, y, z) に展開される
