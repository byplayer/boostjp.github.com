BOOST_PP_BOOL マクロはオペランドに対しての boolean 変換として機能する。

Usage

BOOST_PP_BOOL(x)

Arguments

x
変換される値。 有効な値の範囲は 0 から BOOST_PP_LIMIT_MAG まで。

Remarks

x が 0 ならば、このマクロは 0 に展開される。 そうでなければ、1 に展開される。

See Also

  • BOOST_PP_LIMIT_MAG

Requirements

Header:  <boost/preprocessor/logical/bool.hpp>

Sample Code

#include <boost/preprocessor/logical/bool.hpp>

BOOST_PP_BOOL(6) // 1 に展開される
BOOST_PP_BOOL(0) // 0 に展開される
